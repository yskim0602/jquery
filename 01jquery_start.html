01제이쿼리 소개와 기초문법 익히기

제이쿼리란? 자바스크립트의 생산성을 향상시켜주는 자바스크립트의 라이브러리언어이다. 자바스크립트 라이브러리란? 자주 사용하는 코드들을 재사용할 수 있는 형태로 가공하여 프로그래밍의 효율성을 높여주도록 만든 코드들의 집합이다.

1. 제이쿼리의 특징

  • 앨리먼트(html태그 요소)를 선택하는 강력한 기능을 제공한다.(css선택자 보다 많음)
  • 선택된 앨리먼트를 제어할 수 있는 다양한 함수를 제공한다.
  • 자바스크립트의 자주 사용하는 기능을 라이브러리화한 언어이다.

2. 제이쿼리 라이브러리 연결방법

  • jQuery라이브러리 파일을 직접 연결하기 - 제이쿼리 공식사이트에서 라이브러리 파일을 다온로드 받아서 html문서의 head요소 안에 직접연결하여 사용
  • CDN(Content Delivery Network)주소로 연결하기 - 서비스를 제공하는 업체로부터 링크주소를 사용해서 문서에 연결하여 사용 `제이쿼리 문법(작성법)

3. 제이쿼리 문법(작성법)

  • $('대상').메서드();
  • jquery('대상').메서드();

4. 제이쿼리에서 '$'의 역할

제이쿼리에서는 '$'기호를 맨앞에 작성하는데 이것은 괄호() 안에 넣을 대상을 선택하기 위함이다. 예) $(document),$('.txt01'),$('body'),$('window').........

5. 제이쿼리에서 '$'다음으로 들어가는 것은?

  • html태그요소
  • css선택자
  • 자바스크립트 객체(document.window......)

6. 제이쿼리 기본 wrapper(선언하기)

  • $(document).ready(function(){ 실행할 내용;})
  • $(function() { 실행할 내용;})
  • jQuery(function() { 실행할 내용;})
  • $(function(){실행할 내용;})(jQuery)
